# Programming-Assig# Assignment 1 - C++ Conditional Statements

## Overview
This repository contains solutions for Assignment 1 written in C++. The programs demonstrate the use of conditional statements (`if`, `if-else`, and `else-if`) to solve different real-world problems.

## Programs Included

1. **Value Assignment**
   - Assigns the value of `x` based on the value of `y`.

2. **Weight Calculator**
   - Calculates an object's weight using the formula:
     ```
     Weight = Mass × 9.8
     ```
   - Displays whether the object is too heavy or too light.

3. **Sales Bonus Calculator**
   - Adds a 25% bonus and an additional 250 units if sales exceed 50,000.

4. **Loan Qualification**
   - Checks whether a person qualifies for a special loan based on annual income and years at the current job.

5. **TV Model Features**
   - Displays the features available for TV models 100, 200, and 300.

6. **Magic Date Checker**
   - Determines whether a date is a Magic Date by checking if:
     ```
     Month × Day = Year
     ```

## Concepts Used
- C++
- Input and Output (`cin`, `cout`)
- Variables and Data Types
- Arithmetic Operators
- Relational Operators
- Logical Operators
- Conditional Statements (`if`, `if-else`, `else-if`)
